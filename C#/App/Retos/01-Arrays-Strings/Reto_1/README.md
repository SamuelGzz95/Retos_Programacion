## **1. Two Sum**
- **Descripción:** Dado un array de enteros `nums` y un entero `target`, devuelve los índices de los dos números que sumen `target`.
- **Ejemplo:** 
  - Input: `nums = [2,7,11,15], target = 9`
  - Output: `[0,1]` (porque nums[0] + nums[1] = 9)
- **Restricciones:** Cada input tiene exactamente una solución, no puedes usar el mismo elemento dos veces
- **Complejidad esperada:** O(n) tiempo, O(n) espacio
