## **6. Best Time to Buy and Sell Stock**
- **Descripción:** Encuentra el máximo profit comprando y vendiendo una acción. Debes comprar antes de vender.
- **Ejemplo:**
  - Input: `[7,1,5,3,6,4]`
  - Output: `5` (comprar en día 2 a precio 1, vender en día 5 a precio 6)
- **Restricciones:** 1 <= prices.length <= 10^5
- **Complejidad esperada:** O(n) tiempo, O(1) espacio
