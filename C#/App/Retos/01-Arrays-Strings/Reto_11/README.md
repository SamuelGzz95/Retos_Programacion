## **11. Trapping Rain Water**
- **Descripción:** Dado un array representando elevaciones de barras, calcula cuánta agua de lluvia puede ser atrapada.
- **Ejemplo:**
  - Input: `[0,1,0,2,1,0,1,3,2,1,2,1]`
  - Output: `6`
- **Restricciones:** 1 <= height.length <= 2 * 10^4
- **Complejidad esperada:** O(n) tiempo, O(1) espacio (two pointers)
