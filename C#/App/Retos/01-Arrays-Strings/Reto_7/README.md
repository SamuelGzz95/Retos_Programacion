## **7. Valid Parentheses**
- **Descripción:** Determina si una string con paréntesis `()[]{}` es válida (bien balanceada y en orden correcto).
- **Ejemplo:**
  - Input: `"([])"`
  - Output: `true`
  - Input: `"([)]"`
  - Output: `false`
- **Restricciones:** 1 <= s.length <= 10^4
- **Complejidad esperada:** O(n) tiempo, O(n) espacio
