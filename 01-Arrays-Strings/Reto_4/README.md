## **4. Valid Anagram**
- **Descripción:** Determina si dos strings son anagramas (contienen las mismas letras con las mismas frecuencias).
- **Ejemplo:**
  - Input: `s = "anagram", t = "nagaram"`
  - Output: `true`
- **Restricciones:** Strings contienen solo letras minúsculas
- **Complejidad esperada:** O(n) tiempo, O(1) espacio
