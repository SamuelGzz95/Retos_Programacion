## **15. Regular Expression Matching**
- **Descripción:** Implementa regex matching con soporte para '.' (cualquier char) y '*' (0+ del carácter anterior).
- **Ejemplo:**
  - Input: `s = "aa", p = "a*"`
  - Output: `true`
- **Restricciones:** 1 <= s.length <= 20, 1 <= p.length <= 20
- **Complejidad esperada:** O(m*n) tiempo y espacio (DP)
