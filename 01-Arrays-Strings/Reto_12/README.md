## **12. Longest Substring Without Repeating Characters**
- **Descripción:** Encuentra la longitud de la subcadena más larga sin caracteres repetidos.
- **Ejemplo:**
  - Input: `"abcabcbb"`
  - Output: `3` ("abc")
- **Restricciones:** 0 <= s.length <= 5 * 10^4
- **Complejidad esperada:** O(n) tiempo, O(min(m,n)) espacio (m = tamaño del charset)
