## **8. Maximum Subarray**
- **Descripción:** Encuentra el subarray contiguo con la suma más grande.
- **Ejemplo:**
  - Input: `[-2,1,-3,4,-1,2,1,-5,4]`
  - Output: `6` (subarray [4,-1,2,1])
- **Restricciones:** 1 <= nums.length <= 10^5
- **Complejidad esperada:** O(n) tiempo, O(1) espacio (Kadane's Algorithm)
