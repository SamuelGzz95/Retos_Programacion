## **13. Minimum Window Substring**
- **Descripción:** Encuentra la subcadena mínima de s que contiene todos los caracteres de t (incluyendo duplicados).
- **Ejemplo:**
  - Input: `s = "ADOBECODEBANC", t = "ABC"`
  - Output: `"BANC"`
- **Restricciones:** m, n <= 10^5
- **Complejidad esperada:** O(m+n) tiempo, O(m+n) espacio
