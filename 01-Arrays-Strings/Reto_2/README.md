## **2. Valid Palindrome**
- **Descripción:** Verifica si una string es palíndromo, considerando solo caracteres alfanuméricos e ignorando mayúsculas/minúsculas.
- **Ejemplo:**
  - Input: `"A man, a plan, a canal: Panama"`
  - Output: `true`
- **Restricciones:** 1 <= s.length <= 2 * 10^5
- **Complejidad esperada:** O(n) tiempo, O(1) espacio
