## **17. Sliding Window Maximum**
- **Descripción:** Dado un array y un tamaño de ventana k, encuentra el máximo en cada ventana deslizante.
- **Ejemplo:**
  - Input: `nums = [1,3,-1,-3,5,3,6,7], k = 3`
  - Output: `[3,3,5,5,6,7]`
- **Restricciones:** 1 <= nums.length <= 10^5, 1 <= k <= nums.length
- **Complejidad esperada:** O(n) tiempo (usando deque)
